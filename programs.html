Программы

Программы и утилиты, предназначенные как для совместного использования с различными игровыми движками, так и отдельно от них. Разработка вспомогательных программ ведётся в основном для игровых движков, у которых нет своих инструментов, облегчающих создание игр на данных движках.

Love2D Exporter

Love2D Exporter - программа, предназначенная для портирования на различные платформы проектов, созданных на игровом движке LÖVE.

Pipmak Assistant

Pipmak Assistant - GUI оболочка для игрового движка Pipmak. Данная программа разрабатывалась для простого управления проектами, созданными на этом кроссплатформенном движке.

Guitar Chords

Guitar Chords - База данных аккордов для гитары. С момента существования вышло 4 версии программы. Первая и последняя версия были созданы когда начинал изучать Delphi и Qt соответственно. В последней версии был добавлен простейший генератор аккордов. Программа создавалась для личного использования когда только начинал обучаться игре на гитаре и на звание "профессиональной" никак не претендует.

Polar Studio

Polar Studio - среда разработки для 2D игр. Имеет некоторое сходство с популярным игровым движком Unity3D за счёт похожей реализации скриптинга, системы компонентов и prefab'ов.
