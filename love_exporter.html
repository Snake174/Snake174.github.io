LÖVE Exporter

LÖVE Exporter - программа, предназначенная для портирования на различные платформы проектов, созданных на игровом движке LÖVE.

Возможности программы:

  • экспорт под Windows x86 и x64
  • экспорт под Android 2.3.3, 3.1 и 4.0.3
  • экспорт под Linux

В планах:

  • добавить экспорт под Mac OS
  • возможно, добавить экспорт под Flash или Web
  • добавить проверку обновлений программы
  • добавить возможность встраивания рекламы от AdMob

Для экспорта под андроид необходимо, чтобы был установлен Android SDK и соответствующие версии API (10, 12 и 15) и самая последняя версия API (Android 4.4.2 API 19).

Поддерживаемые версии Linux:

  • Ubuntu 12.04+
  • Fedora 20+
  • Linux Mint 13+
  • openSUSE 13.1+

Для корректной работы программы также необходимо, чтобы пути до файлов не содержали пробелов и русских букв.

Хочу отдельно поблагодарить пользователей форума GCUP silver52rus и burlachenko за то, что активно помогали в тестировании программы.

Проверка созданных apk файлов для Android проводилась на BlueStacks, Genymotion, телефоне Samsung Duos (Android 2.6), Mediox Mid9742 DC (Android 4.0), Asus PhonePad (Android 4) и Asus MemoPad (Android 4.4.2).
