Love2D-Helpers

Love2D-Helpers представляет собой набор вспомогательных утилит для разработки на игровом движке Love2D.

Текущая поддерживаемая версия 0.10.1

Скрипты написаны на Python 3 и PyQt4.

SDK предназначен только для Windows x64
