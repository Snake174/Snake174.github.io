Полезные ссылки

В данном разделе будут публиковаться все полезные ссылки, которые могут Вам когда-нибудь пригодиться.

GcUp

Портал, посвящённый созданию игр, в том числе и без программирования. На сайте можно скачать необходимые для этого инструменты (конструкторы игр, движки), изучить необходимую литературу (статьи, журналы, книги) и т.д. Здесь также имеется форум, где Вы всегда сможете спросить совет или поделиться своими наработками.

FreeSound

Сайт с бесплатной музыкой и разного рода звуками. Присутствует возможность предварительного прослушивания композиций. Для скачивания файлов необходима регистрация.

Love2D

Сайт, посвящённый игровому движку LÖVE. Движок предназначен для разработки 2D игр на языке программирования LUA. Игры, созданные при помощи данного движка работают на платформах Windows, Mac OS X и Linux. Также ведётся портирование на другие платформы, такие как Android, Web и другие. На форуме можно задать вопрос по данному игровому движку и делиться своими наработками.

Pipmak

Сайт, посвящённый игровому движку Pipmak. Pipmak - кроссплатформенный игровой движок с открытым исходным кодом, предназначенный для создания point-and-click квестов. Также можно создавать игры с панорамным обзором по типу игры Myst III: Exile. Используется язык программирования Lua.

Перейти

Menucool

Сайт для web разработчиков. Здесь можно посмотреть как делаются меню, всплывающие подсказки, вкладки, галерея изображений и многое другое. Используется как чистый CSS, так и JavaScript. Можно скачать примеры.

D.I.E.S Game Studio

D.I.E.S GS - российская компания, специализирующаяся на разработке компьютерных игр на игровом движке Unity3D. Данная компания основана совсем недавно и было принято решение заниматься только MMO проектами.
