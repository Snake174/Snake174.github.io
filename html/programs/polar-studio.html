Polar Studio

Polar Studio - среда разработки для 2D игр, использующая OpenGL для вывода графики и C# в качестве скриптового языка. Работает на платформе Windows 7 и 8.

Благодаря данному редактору вы с лёгкостью можете создавать игры в 2D пространстве с использованием шейдеров и анимации на языке программирования C#. Polar Studio имеет некоторое сходство с популярным игровым движком Unity3D за счёт похожей реализации скриптинга, системы компонентов и prefab'ов. Также имеется отладчик, который можно подключить в режиме Debug, с помощью которого контроль за разработкой игр становится ещё проще.

В новых версиях в редактор был встроен физический движок Box2D и улучшен механизм подачи данных в шейдер (glsl). Также были введены материалы и усовершенствована работа с камерами. Теперь всё стало намного проще и эффективнее, чем было раньше. Все ресурсы игры хранятся в специальных файлах со своим форматом, что делает попытку обычного их копирования весьма затруднительной. Для корректной работы программы необходимо установить .Net Framework 4.

Данная программа находится в стадии разработки. Разрабатывается пользователем форума GcUp tracer07. Пожалуйста, тестируйте движок и пишите результаты. Автор будет благодарен за конструктивную критику и пожелания.

Возможности / особенности:

  • Есть редактор, своя среда разработки
  • Шейдеры
  • Анимация
  • Возможность отладки приложений
  • C# в качестве скриптового языка
  • Unity-подобная система компонентов
  • Unity-подобный скриптинг

Скриншоты

Версия 1.01а:

  • Ресурсы сгруппированы по папкам
  • Реализована система компонентов почти как в Unity3D
  • Реализован скриптинг почти как в Unity3D
  • Добавлены режимы Debug и Release для сборки проекта (в Debug можно присоединять отладчик)
  • Реализована система prefab'ов
  • Можно назначать иконку приложению и стартовую сцену
  • Общее повышение производительности
  • Добавлена консоль. Теперь все сообщения редактора выводятся туда

Версия 1.02а:

  • Добавлена возможность приближать и отдалять камеру
  • Добавлена возможность назначать переменные в редакторе (как в Unity3D)
  • Добавлена простенькая демка показывающая вращение планет вокруг солнца
