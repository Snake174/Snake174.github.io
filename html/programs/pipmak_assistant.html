Pipmak Assistant

Pipmak Assistant - среда разработки для игрового движка Pipmak. Данный игровой движок идеально подходит для создания point-and-click квестов. Также можно создавать игры с панорамным обзором по типу игры Myst III: Exile. Игры, созданные при помощи Pipmak работают на платформах Windows, Linux и Mac OS. Используется скриптовый язык программирования Lua.

Для разработки игр на данном движке всё приходится делать вручную. Иначе говоря, для каждой локации необходимо создать папку, в которой будут размещены ресурсы игры. Также в этой папке должен присутствовать Lua файл, в котором задаётся логика игрового процесса. Для локаций нужно подготовить файл хотспотов, определяющий порядок перехода между ними и шесть изображений кубической панорамы, не считая звуковых эффектов и другой необходимой информации.

Для пары-тройки локаций весь процесс подготовки кажется вполне нормальным, но представьте, если их будет гораздо больше. После 5 - 10 таких манипуляций возникает желание разбить вдребезги клаву о монитор. Естесственно, что данное занятие быстро надоедает и отвлекает от непосредственного создания самой игры.

Данная IDE как раз и разрабатывалась для более удобного управления проектами на движке Pipmak. Программа берёт всю рутинную работу на себя, что меньше отвлекает вас от разработки и воплощения интересных идей.

Возможности программы:

  • загрузка проектов и построение структуры проекта в виде дерева
  • просмотр кубической панорамы
  • просмотр связей между нодами
  • просмотр и изменение ресурсов движка
  • запуск проекта при помощи плеера Pipmak
  • просмотр слайдов
  • печать связей нодов и сохранение в файл
  • проигрывание ogg файлов
  • просмотр хотспотов
  • обновление программы
