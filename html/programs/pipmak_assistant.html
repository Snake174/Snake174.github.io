Pipmak Assistant

Описание готовится...

Возможности программы:

  • загрузка проектов и построение структуры проекта в виде дерева
  • просмотр кубической панорамы
  • просмотр связей между нодами
  • просмотр и изменение ресурсов движка Pipmak
  • запуск проекта при помощи плеера Pipmak
  • просмотр слайдов
  • печать связей нодов и сохранение в файл
  • проигрывание ogg файлов
  • обновление программы
  • просмотр хотспотов
